# Memory Match Game

A classic memory matching card game designed to help submariners keep their minds sharp during long deployments.

## Features

### Core Gameplay
- **Flip & Match**: Tap cards to flip them, find matching pairs
- **3D Flip Animation**: Smooth card flip transitions
- **Ocean Theme**: Submarine and ocean-themed icons

### Difficulty Levels
- **Easy**: 3×4 grid (6 pairs)
- **Medium**: 4×4 grid (8 pairs)
- **Hard**: 4×6 grid (12 pairs)

### Progress Tracking
- **Move Counter**: Track how many moves you make
- **Timer**: See how fast you can complete the puzzle
- **Personal Bests**: Saves best moves and time for each difficulty
- **Match Counter**: Real-time progress indicator

### Visual & Audio Feedback
- **Sound Effects**: Tap sounds and success chimes
- **Haptic Feedback**: Vibration on matches
- **Color-Coded Cards**: Each pair has its own unique color
- **Match Animations**: Cards fade when matched

### Mental Health Benefits
- **Memory Training**: Proven cognitive exercise
- **Focus & Concentration**: Requires sustained attention
- **Stress Relief**: Engaging distraction from stress
- **Sense of Achievement**: Clear goals and completion rewards
- **No Pressure**: Can be played at own pace

## Game Rules

1. Start a new game at any difficulty level
2. Tap any card to flip it over
3. Tap a second card to try to find a match
4. If cards match, they stay face up
5. If they don't match, they flip back over
6. Remember card positions to find matches faster
7. Complete the game by matching all pairs

## Controls

- **Tap Card**: Flip a card
- **New Game Button**: Start over with shuffled cards
- **Difficulty Buttons**: Switch difficulty (only when game not started)
- **Back Button**: Return to home screen

## Implementation Details

### Files
- `models/memory_card.dart`: Card data model and difficulty levels
- `widgets/memory_card_widget.dart`: Animated card component
- `screens/memory_match_screen.dart`: Main game screen and logic

### Key Features
- Random card shuffling for each new game
- Prevents flipping more than 2 cards at once
- Tracks best scores per difficulty level
- Responsive grid layout for different screen sizes
- Clean, simple UI with no distractions

1. **Proven Benefits**: Memory games have well-researched cognitive benefits

4. **Offline-First**: No internet required, works completely offline
